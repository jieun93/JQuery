window 객체

- 브라우저 모델 객체 (BOM)에서 최상위 객체로서 웹 브라우저 윈도우를 나타낸다. - open() / close()
- setInterval() / clearInterval() / setTimeout()
- alert() / confirm() / prompt()
- focus() - 속성 : opener

window객체

- setTimeout(function, millisec): 정해진 시간 후에 function을 1번만 실행한다. - setInterval(function, millisec):정해진 시간 간격으로 매번 function을 수행 - clearInterval(변수) : setInterval에 의해서 실행된 기능을 중지
